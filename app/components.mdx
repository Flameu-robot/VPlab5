# Button Component

Кнопка с различными вариантами стилей, размерами и состояниями.

## Props

| Prop    | Type                                 | Default   | Description            |
|---------|--------------------------------------|-----------|------------------------|
| variant | 'primary' \| 'secondary' \| 'danger' | 'primary' | Стиль кнопки           |
| size    | 'sm' \| 'md' \| 'lg'                 | 'md'      | Размер кнопки          |
| disabled| boolean                              | false     | Отключено ли состояние |
| onClick | () => void                           | -         | Обработчик клика       |
| children| ReactNode                            | -         | Содержимое кнопки      |
| type    | 'button' \| 'submit' \| 'reset'      | 'button'  | Тип кнопки             |
